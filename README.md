# Online Auction System (MySQL)

A database-driven auction platform designed as a course term project.  
This project focuses on relational schema design, core auction business logic, and SQL implementation.

## Features
- User & item management
- Auction listing and bidding workflow
- Transaction/payment and shipping records
- Relational integrity with primary/foreign keys

## Tech Stack
- MySQL
- SQL
- ER modeling (draw.io)
